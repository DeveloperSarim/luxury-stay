# Luxury Stay - Frontend

Hotel Management System Frontend built with React and Vite.

## Features

- User Dashboard
- Admin Dashboard
- Reception Dashboard
- Room Booking
- Live Chat
- QR Code Scanner (Camera-based)
- Responsive Design
- Modern UI/UX

## Tech Stack

- React 19
- Vite
- React Router
- GSAP (Animations)
- HTML5 QR Code Scanner
- CSS3

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Environment Variables

Create `.env` file:
```env
VITE_API_URL=https://luxury-stay-backend.vercel.app
```

## Project Structure

```
src/
├── components/     # Reusable components
├── views/         # Page components
├── context/       # React context
├── utils/         # Utility functions
└── router.jsx     # Route configuration
```
